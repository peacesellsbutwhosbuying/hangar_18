Проект «Hangar 18»

1.	Описание проекта
Название: Hangar 18
Назначение проекта:  В готовом виде проект будет предназначен для развлечения.


Краткое описание: Игра 2D платформер в стиле 8 или 16 бит, написанная на языке программирования Python, используя библиотеку для разработки игр PyGame.
